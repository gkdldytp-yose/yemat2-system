### 1️⃣ 프로젝트 폴더 열기
다운받은 `yemat1` 폴더를 찾으세요.

### 2️⃣ Flask 설치 (최초 1회만)

**Windows - 명령 프롬프트에서:**
```cmd
pip install flask
```

**Mac/Linux - 터미널에서:**
```bash
pip3 install flask
```

### 3️⃣ 서버 실행

**Windows:**
- `run.bat` 파일 더블클릭
- 또는 명령 프롬프트에서 `python app.py`

**Mac/Linux:**
- 터미널에서 `./run.sh`
- 또는 `python3 app.py`

### 4️⃣ 브라우저에서 접속
```
http://localhost:5000
```

### 5️⃣ 로그인

## ❗ 자주 묻는 질문

**Q: Flask가 설치 안 돼요**
```bash
python3 -m pip install flask --user
```

**Q: python 명령어를 찾을 수 없어요**
- Python을 설치하세요: https://www.python.org/downloads/
- 설치 시 "Add Python to PATH" 체크!

**Q: 5000번 포트가 이미 사용 중이래요**
- `app.py` 파일에서 `port=5000`을 `port=8000`으로 변경
- 그리고 `http://localhost:8000` 접속

**Q: 로그인 화면이 안 나와요**
1. 서버가 실행 중인지 확인 (터미널에 메시지 있어야 함)
2. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
3. `http://127.0.0.1:5000` 으로도 시도
